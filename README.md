# BridgePay — Cross-Border Payments for the Unbanked

## What is BridgePay?

BridgePay is a BA portfolio project I built to solve a real problem —
700 million unbanked migrant workers paying 6–10% fees to send money
home, with transfers taking 3–5 days.

This project documents the full BA lifecycle for a mobile-first
remittance platform that cuts fees below 2% and settles transfers
in real time — no bank account required.


## The Problem

A Filipino domestic worker in Dubai sends $300 home every month.
She pays $18 in fees. The money takes 3 days. Her family waits.

This happens 700 million times a month globally.

The technology to fix this exists. The problem is that every
digital solution requires a bank account — which these workers
don't have.

BridgePay is built specifically for them.

## Key Highlights

- **Market:** $857B global remittance market
- **Problem:** 42% of senders are unbanked, paying avg 6.2% fees
- **Solution:** Mobile-first, agent-assisted, KYC-lite remittance platform
- **Target Corridors (Phase 1):** UAE→Philippines, USA→Mexico, KSA→India
- **Fee Target:** Under 2% vs industry average of 6.2%
- **SDG Alignment:** SDG 1, SDG 10, SDG 17
